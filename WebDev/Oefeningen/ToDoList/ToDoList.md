# ToDoList

Een veel voorkomende webapplicatie is er een waar je een lijst bijhoud van elementen en deze afbeeld.
Vervolgens kan je nieuwe elementen toevoegen aan de lijst en ze ook terug verwijderen

# Opdracht

Maak een webpagina met een textinput en een Add-knop.
Wanneer op de Add-knop geklikt wordt voeg je de inhoude van de textinput toe in een lijst en beeld je deze af in de webapplicatie.

Vervolgens voeg je bij elk element een kruisje toe.
Wanneer de gebruiker op dit kruisje klikt wordt het element verwijderd uit de lijst.

# Bonus

Zorg ervoor dat je de inhoud van de elementen kan aanpassen
