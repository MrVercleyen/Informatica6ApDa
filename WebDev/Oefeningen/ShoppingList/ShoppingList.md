# ToDoList

Een veel voorkomende webapplicatie is er een waar je een lijst bijhoud van elementen en deze afbeeldt.
Vervolgens kan je nieuwe elementen toevoegen aan de lijst en ze ook terug verwijderen.

Wij gaan dit ontwerpen in de vorm van een shoppinglist met ingrediënten.

# Opdracht

1. Maak een webpagina met een textinput, twee numberinputten en een Add-knop.
2. Wanneer op de Add-knop geklikt wordt voeg je het ingrediënt toe aan de shoppinglist met het aantal goederen en de prijs op de webpagina.
3. Vervolgens voeg je bij elk element een kruisje toe. Wanneer de gebruiker op dit kruisje klikt wordt het element verwijderd uit de lijst.
4. Hou zeker een lijst bij in de javascriptapplicatie met de toegevoegde ingrediënten. Deze ingrediënten zullen in de vorm zijn van objecten met een aantal properties.
   !TIP! : Ontwerp eerst het object voor je aan deze oefening begint.
5. Bereken het totaal van de prijs van alle ingrediënten en beeld deze af op de webpagina.

# Bonus

Zorg ervoor dat je de ingrediënten ook kan aanpassen
