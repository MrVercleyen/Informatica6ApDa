# Dynamisch Vierkant

Voor deze opdracht ontwerpen jullie een vierkant met een aantal parameters die jullie kunnen aanpassen aan de hand van een webinterface.

# Opdracht

Maak een nieuwe webpagina met een eenvoudige div en geef die een rode kleur.
Zorg ervoor dat je een Width en een Height meeheeft aan het vierkant.

Voorzie inputvelden op de webpagina waarmee je volgende properties kan aanpassen:

    - Width
    - Height
    - BorderWidth
    - BorderRadius
    - Color (van het vierkant)
