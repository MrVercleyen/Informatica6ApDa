# Mover

Met Javascript kunnen we ook simpele games namaken.
Aan de hand van javascript code kunnen we keyevents opvangen en vervolgens elementen gaan aanpassen.

# Opdracht

Maak een nieuwe html pagina aan en plaats deze code in de body-tag

     <div>
      <div
        id="mover"
        style="
          background-color: red;
          width: 100px;
          height: 20px;
          position: relative;
        "
      ></div>
    </div>

Nu hebben we een kleine balk in de linkerbovenhoek van ons scherm
Zorg er nu voor met de 'onKeyDown' event dat we de balk kunnen verplaatsen met de pijltjestoetsen.

Doe dit door de Top & Left aan te passen van het element.

Elke toets heeft een eigen keyCode, zoek op het internet wat de codes zijn voor de pijltjestoetsen.
