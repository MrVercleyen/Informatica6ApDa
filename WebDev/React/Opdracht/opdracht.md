# CMS App

Nadat je de juiste API request hebt afgehandeld en een script hebt doorgestuurd die de juiste informatie ophaald, kan je starten met het maken van een React App om de informatie af te beelden.

## Opdracht

Schrijf een React App met volgende element :

    - Een Get request die de info ophaalt van de CMS
    - Een component Persoon
    - Een component Bedrijf
    - Een pagina met een overzicht van de personen
    - Een pagina met een overzicht van de bedrijven
    - Een detailpagina voor een persoon
    - Een detailpagina voor een bedrijf
    - mogelijkheid om van de persoon naar een bedrijf te navigeren en omgekeerd

## Indienen

Stuur de react applicatie in een zip door naar Vital Vercleyen op Smartschool
