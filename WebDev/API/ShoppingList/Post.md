# Opdracht POST

Voor het vervolledigen van de Full-Stack Applicatie moeten we enkel nog de json-file aanpassen bij wijzigingen in de frontend.

# Opdracht Stap voor stap

- Voer eerst en vooral het command 'npm install' uit in de map van ShoppingListAPI
  - Dit genereert dan de map 'node_modules' en de file 'package-lock.json'

- Plaats de inhoud van de lijst in de post-request in de savelist() methode
- Haal de inhoud van de post request op in de backend via de body
- sla de inhoud van de body op in de json file via de fs-module
