# CMS

Zoek een CMS met volgende vereisten:

    - Gratis gebruik
    - Informatie opvragen via API

## Content toevoegen

- Voeg 2 bedrijven toe.
  - Naam
  - Adres
  - Type
  - ...
- Voeg 10 contactpersonen toe en verdeel deze onder de 2 bedrijven
  - Voornaam
  - Familienaam
  - Gsm
  - Adres
  - Leeftijd
  - ...

## Content ophalen

Zorg dat je de toegevoegde data kan ophalen aan de hand van een GET-request naar de API van jouw gekozen CMS
