# Probleemsituatie Cloud

Voor deze opdracht krijg je een specifieke probleemsituatie die je moet oplossen met de functionaliteiten van een Cloud Platform.

## Instructies

- Werk stapsgewijs en gestructureerd.
- Lijst ALLE stappen die je uitvoert duidelijk op, van begin tot einde.
- Beschrijf per stap:
  - wat je doet
  - waarom je het doet
  - welke tool of service je gebruikt
- Voeg waar mogelijk schema’s, screenshots of diagrammen toe.
- Denk na over:
  - schaalbaarheid
  - veiligheid (wie heeft toegang tot de data?)
  - kost (gebruik van cloud resources)

## Probleemstelling

De gemeente wil de luchtkwaliteit rond scholen beter opvolgen. Ze hebben sensoren geplaatst die continu data verzamelen (zoals CO₂, fijnstof en temperatuur), maar momenteel gebeurt er niets met deze data.

De opdracht is om een systeem te ontwikkelen dat:

- de sensordata automatisch verzamelt en opslaat in de cloud
- de data verwerkt en analyseert
- waarschuwingen geeft wanneer waarden te hoog zijn
- de resultaten visualiseert in een dashboard voor de schooldirectie

Hiervoor moet je gebruik maken van een cloudplatform zoals Google Cloud Platform.

## Elementen

Ontwikkel een oplossing waarbij je verschillende cloudfunctionaliteiten combineert, zoals:

- data-opslag
- data-verwerking
- automatisatie
- visualisatie

## Extra

- Ontwikkel een functionaliteit die een mail stuurt wanneer de waarden een bepaalde waarde overstijgen
