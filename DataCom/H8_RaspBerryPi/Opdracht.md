# Project RaspBerryPi

Een RaspBerryPi is een kleine computer waar je makkelijk software op kan runnen.
We hebben al kort een introductie gezien hoe we werken met een RaspBerryPi, maar nu is het aan jullie.

Jullie gaan te werk in groepjes van 2 of 3 personen en kiezen samen wat jullie willen ontwikkelen.
Wat jullie juist maken mag je volledig zelf kiezen maar er zijn een aantal minimumvereisten.

## Minimumvereisten

- Werk met een systeem om data op te slaan. Dit mag een database zijn maar ook een simpele txt file.
- Voorzie een webapplicatie om gegevens te displayen. Dit kan gaan over logs, statistieken, login, ...

## Start

Om van start te gaan met jullie project wil ik dat jullie eerst en vooral goed nadenken over wat jullie nodig hebben en welke technologiën jullie gaan gebruiken.

Shrijf een document uit met een korte omschrijving van jullie eindproduct.
Noteer ook welke stappen jullie zullen ondernemen om het eindproduct te bereiken.
Denk goed na over welke technologiën jullie gaan gebruiken en welke hardware jullie nodig hebben, plaats dit ook in het document.
